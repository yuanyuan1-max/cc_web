# Claude Code Agent 全栈架构手册

> 内核级源码解析 · 节点化架构解构 · 技术细节全量收录

一份基于真实源码提炼的 Claude Code Agent 操作系统深度白皮书。不仅仅是文档，更是对 AI Agent 运行内核的完整探索。

## 📚 核心板块

- **Agent 全流程图** - 从输入解析到递归工具调用的完整异步生成器状态机
- **核心引擎拓扑** - 解密 Coordinator、Fork 机制与底层 Sandbox 的交互拓扑
- **记忆系统深度解析** - MEMORY.md 截断、Drift 防护与加载优先级的实现算法
- **提示词生产手册** - 对抗性验证、System Prompt 动态边界等顶级提示词
- **工具调用协议** - str_replace 唯一性算法与 Tool Result 预算控制
- **斜线命令参考** - 全量 60+ 斜线命令功能说明、用法示例与源码对应
- **工程化设计** - 确定性执行、快速失败、资源预算、可观测性、隔离与沙箱
- **ANT 门控链路** - USER_TYPE=ant 条件门控对 ConfigTool 等能力的上下游影响
- **术语与路径** - 源码全路径索引，扫清所有模糊概念

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 `http://localhost:5177` 查看手册。

### 生产构建

```bash
npm run build
```

## 📖 内容特色

### 1. 节点化架构解构
每个核心模块都通过节点图展示上下游关系，清晰呈现数据流与控制流。

### 2. 源码级深度解析
所有技术细节都对应真实源码路径，可直接追溯到实现。

### 3. 工程化思维体现
涵盖确定性执行、快速失败、资源预算、可观测性、隔离防护等现代系统设计原则。

中英文对照，便于国际化理解。
